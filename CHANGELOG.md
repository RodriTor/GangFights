# Registro de cambios

## [1.0.0] - 2026-07-16
### Añadido - Estructura del juego
- Estructura base del videojuego generada mediante gdx-liftoff.
- Soporte integrado de Java 21 y LibGDX v1.14.
- Activación de dependencias y extensiones: Freetype, Box2D, AI, Controllers y Box2D Lights.
- Creación de archivos de control y documentación: `.gitignore`, `README.md` y `CHANGELOG.md`.
